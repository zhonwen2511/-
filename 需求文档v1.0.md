# 校园二手书交易平台 - 需求文档 v1.0

## 1. 项目概述

### 1.1 项目背景
随着高校学生数量的不断增加，二手书籍交易市场潜力巨大。校园二手书交易平台旨在为在校师生提供一个便捷、安全的二手书籍交易渠道，帮助学生节省购书成本，实现书籍资源的循环利用。

### 1.2 项目目标
- 为校园用户提供一个便捷的二手书买卖平台
- 实现书籍信息的快速发布与检索
- 提供安全的交易保障机制
- 促进校园书籍资源的有效循环

### 1.3 项目类型
- **项目类型**: Web 应用（前后端分离）
- **技术栈**: Vue.js (前端) + Node.js (后端)
- **数据库**: SQLite (本地数据库)

## 2. 功能模块

### 2.1 用户系统

| 功能 | 描述 |
|------|------|
| 用户注册 | 支持邮箱/手机号注册，设置用户名和密码 |
| 用户登录 | 使用账号密码登录，JWT token 认证 |
| 个人信息 | 查看和编辑个人资料（头像、昵称、联系方式） |
| 我的发布 | 查看自己发布的所有书籍 |

### 2.2 书籍发布

| 功能 | 描述 |
|------|------|
| 发布书籍 | 填写书籍信息（书名、作者、ISBN、分类、价格、成色、描述、图片） |
| 编辑书籍 | 修改已发布书籍的信息 |
| 删除书籍 | 下架/删除已发布的书籍 |
| 上传图片 | 支持上传书籍封面图片 |

### 2.3 书籍搜索

| 功能 | 描述 |
|------|------|
| 关键词搜索 | 按书名、作者、ISBN 模糊搜索 |
| 分类筛选 | 按书籍分类筛选（教材、文学、专业书、考试资料等） |
| 价格排序 | 按价格升序/降序排列 |
| 最新发布 | 按发布时间排序 |

### 2.4 浏览分类

| 功能 | 描述 |
|------|------|
| 分类目录 | 教材类、文学类、专业书、考试资料、生活百科、其他 |
| 书籍列表 | 展示书籍封面、名称、价格、卖家信息 |
| 书籍详情 | 查看书籍详细信息 |

### 2.5 购物车与订单

| 功能 | 描述 |
|------|------|
| 加入购物车 | 将感兴趣的书加入购物车 |
| 购物车管理 | 查看、删除购物车中的书籍 |
| 下单 | 确认订单信息并提交 |
| 订单管理 | 查看我的订单（待付款、已完成） |
| 取消订单 | 取消未支付的订单 |

## 3. 数据库设计

### 3.1 用户表 (users)
| 字段 | 类型 | 描述 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| username | TEXT | 用户名，唯一 |
| password | TEXT | 密码（加密存储） |
| email | TEXT | 邮箱 |
| phone | TEXT | 电话 |
| avatar | TEXT | 头像URL |
| created_at | DATETIME | 创建时间 |

### 3.2 书籍表 (books)
| 字段 | 类型 | 描述 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| title | TEXT | 书名 |
| author | TEXT | 作者 |
| isbn | TEXT | ISBN |
| category | TEXT | 分类 |
| price | REAL | 价格 |
| condition | TEXT | 成色（新/旧） |
| description | TEXT | 描述 |
| image | TEXT | 图片URL |
| user_id | INTEGER | 卖家ID |
| status | TEXT | 状态（available/sold） |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

### 3.3 购物车表 (cart)
| 字段 | 类型 | 描述 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| user_id | INTEGER | 用户ID |
| book_id | INTEGER | 书籍ID |
| created_at | DATETIME | 添加时间 |

### 3.4 订单表 (orders)
| 字段 | 类型 | 描述 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| order_no | TEXT | 订单号 |
| user_id | INTEGER | 买家ID |
| book_id | INTEGER | 书籍ID |
| total_price | REAL | 总价 |
| status | TEXT | 状态（pending/paid/completed/cancelled） |
| created_at | DATETIME | 创建时间 |

## 4. API 接口设计

### 4.1 用户模块
| 方法 | 路径 | 描述 |
|------|------|------|
| POST | /api/auth/register | 用户注册 |
| POST | /api/auth/login | 用户登录 |
| GET | /api/auth/profile | 获取用户信息 |
| PUT | /api/auth/profile | 更新用户信息 |

### 4.2 书籍模块
| 方法 | 路径 | 描述 |
|------|------|------|
| GET | /api/books | 获取书籍列表 |
| GET | /api/books/:id | 获取书籍详情 |
| POST | /api/books | 发布书籍 |
| PUT | /api/books/:id | 更新书籍 |
| DELETE | /api/books/:id | 删除书籍 |
| GET | /api/books/search | 搜索书籍 |

### 4.3 购物车模块
| 方法 | 路径 | 描述 |
|------|------|------|
| GET | /api/cart | 获取购物车列表 |
| POST | /api/cart | 加入购物车 |
| DELETE | /api/cart/:id | 移除购物车项 |

### 4.4 订单模块
| 方法 | 路径 | 描述 |
|------|------|------|
| GET | /api/orders | 获取订单列表 |
| POST | /api/orders | 创建订单 |
| PUT | /api/orders/:id | 更新订单状态 |
| DELETE | /api/orders/:id | 取消订单 |

## 6. 非功能性需求

### 6.1 性能要求
- 页面加载时间不超过 3 秒
- API 响应时间不超过 500ms

### 6.2 安全性
- 密码加密存储 (bcrypt)
- JWT token 身份验证
- 输入数据校验与过滤

### 6.3 兼容性
- 支持主流现代浏览器 (Chrome, Firefox, Edge, Safari)
- 响应式布局，适配 PC 和移动端
