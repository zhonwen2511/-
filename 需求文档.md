# 校园二手书交易平台 - 需求文档

## 1. 项目概述

CampusBook 是一个面向高校师生的校园二手书交易平台，旨在帮助学生买卖二手教材和书籍，实现书籍资源的循环利用。

## 2. 技术栈

- **前端**: Vue.js 3
- **后端**: Node.js + Express
- **数据库**: SQLite

## 3. 核心功能

### 3.1 用户系统
- 用户注册/登录
- 个人信息管理
- 我的发布

### 3.2 书籍交易
- 发布书籍（书名、作者、ISBN、分类、价格、成色、描述、图片）
- 编辑/删除书籍
- 书籍搜索（关键词、分类）
- 分类浏览

### 3.3 购物与订单
- 加入购物车
- 下单
- 订单管理

## 4. 数据库

- users 表：用户信息
- books 表：书籍信息
- cart 表：购物车
- orders 表：订单

## 5. API

| 模块 | 接口 |
|------|------|
| 认证 | register, login, profile |
| 书籍 | list, detail, publish, update, delete, search |
| 购物车 | list, add, remove |
| 订单 | list, create, update, cancel |
